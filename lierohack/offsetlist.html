CPU player settings
Offset Field Size Contents
0001AEF4h Change weapon right 1 Short Determines how often the cpu player changes his weapon to the right. 0 = biggest chance.
0001AEF6h Change weapon left 1 Short Determines how often the cpu player changes his weapon to the left. 0 = biggest chance.
0001AEF8h Fire button press 1 Short Determines how often the cpu player fires. 0 = biggest chance.
0001AEFAh Change button press 1 Short Determines how often the cpu player starts changing weapons. 0 = biggest chance.
0001AEFCh Jump button press 1 Short Determines how often the cpu player tries to jump. 0 = biggest chance.
0001AF06h Fire button release 1 Short Determines how often the cpu player stops firing (when firing?). 0 = biggest chance.
0001AF08h Change button release 1 Short Determines how often the cpu player stops changing weapons (when changing weapons?). 0 = biggest chance.
0001AF0Ah Jump button release 1 Short Determines how often the cpu player releases the jump button. 0 = biggest chance.

Material settings


Data blocks are 32 bytes long. 1 byte contains property for 8 colors.
For example: value 240d == 00001111 converted to reversed binary string.
This means the property is set for the last 4 colors in that group of 8 colors.
Offset Field Size Contents
0001C2E0h Is dirt (brown edged) 32 * 1 Byte Acts like dirt: weapon projectiles collide with this matter, weapons can destroy it, ninja rope can attach to it.
0001C300h Is dirt (green edged) 32 * 1 Byte Same as the above accept when digging, the edges have a different color.
0001C320h Is rock 32 * 1 Byte Acts like rock, weapon projectiles collide with this matter, ninja rope can attach to it.
0001C340h Is background 32 * 1 Byte Acts like background: you cannot stand on this. If this is off you can stand on this.
0001C360h Shadows visible 32 * 1 Byte On this color you can see the shadow of rock & dirt (shadow = normal index +4).
