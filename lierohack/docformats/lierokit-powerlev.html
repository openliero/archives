LieroKit Powerlevel (LEV) file format

See also "Liero level (LEV) file format".

Powerlevels are much like normal levels, but they include a new palette. The document layout is quite simple. Powerlevels are always 177178 bytes divided into 3 blocks:

  • 176400 bytes of level data like any normal Liero level.
  • The string "POWERLEVEL". You can use this to identify the file as a Powerlevel.
  • 768 bytes of palette data.
The palette should be an '18-bit': 3 bytes for each color, representing the Red, Green and Blue values ranging from 0 to 63 (!). There are 256 colors and no white spaces in the palette, so there is 3*256=768 bytes of palette data.
