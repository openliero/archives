LieroKit Palette (LPL) file format

This format is exactly the same as the ACT palette format.

The palette is an ordinary 24-bit palette: 3 bytes for each color, representing the Red, Green and Blue values ranging from 0 to 255. There are 256 colors and no white spaces in the palette, so the file is 3*256=768 bytes in size.

Before inserting the palette in LIERO.EXE, the palette should be converted to '18-bit': Also 3 bytes for each color, but the last (first?) two bits are always 0 so the values range from 0 to 63. This is the palette type used by Liero.
